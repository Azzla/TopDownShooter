# TopDownShooter
My first longterm game project. Utilizes rogue-like mechanics/progression. Features custom music, sound effects, and (terrible) pixel art by me.

Written in Lua, using the excellent LOVE2D engine.

Movement: WASD
Shoot: Mouse1
Reload: R
Dash: Spacebar
Pause: Escape
Quit Game: End
